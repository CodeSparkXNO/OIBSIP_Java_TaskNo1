# Online Reservation System (Java Project)

## Internship Task
OIBSIP Java Development Internship – Task 1

## Project Description
The Online Reservation System is a desktop-based Java application developed using Java Swing and MySQL database. The system allows users to login and perform reservation-related operations.

## Features
- User Login Authentication
- New Reservation
- Cancel Reservation
- Database Connectivity (MySQL)
- Simple GUI using Java Swing

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC
- VS Code
- Git & GitHub

## How to Run
1. Install XAMPP and start MySQL
2. Import database using phpMyAdmin
3. Open project in VS Code
4. Run `Login.java`
